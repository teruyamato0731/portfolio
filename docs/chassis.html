About

高専在学中にロボコンで使用するために自作したOSSです。 MITライセンスで公開しています。

Chassisはロボコンでの足回り制御を行うためのC++ライブラリである。 ロボコン用足回りのモータ出力計算と自己位置推定による自動制御をサポートする。

https://teruyamato0731.github.io/Chassis/

Features

  • 三輪オムニや四輪オムニ、メカナム、四輪独立ステアリングなどのモータ出力計算に対応。
  • エンコーダのオドメトリを用いた自己位置推定に対応。
  • 自己位置推定とPID制御による自動制御に対応。 自己位置推定は任意のセンサとフィルタの組み合わせで実装できる。

Supported Environment

  • Platform IO の入った VSCode。もしくは Arduino IDE。
  • C++で動くフレームワークとマイコン。
  • ArduinoやMbedで動くnucleo, esp32, teensyなどに対応。
